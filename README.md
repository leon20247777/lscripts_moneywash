**Money Wash Script** is a FiveM script that allows players to launder money with a **timer**, **staff tiers**, and **Discord logging**. It is fully compatible with **ox_target**, **ox_lib**, and **ox_inventory**, making it easy to integrate into any server setup.

## Features

- **Money Laundering with Timer:** Players must wait while the laundering process completes.  
- **Staff Tiers:** Assign different permissions and roles to staff members.  
- **Discord Logging:** All actions are logged automatically to a Discord channel.  
- **ox_target & ox_lib Integration:** Interact seamlessly with world targets.  
- **ox_inventory Support:** Items are directly managed in the player inventory.  
- **Modular & Customizable:** Easily extendable to suit your server needs.
